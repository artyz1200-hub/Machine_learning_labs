# Machine Learning Labs & Case Studies


A collection of various machine learning projects, benchmarks, and case studies ranging from computer vision to large language models (LLMs).

## 🚀 Projects Included

*   **Benchmarking Lightweight Local LLMs:** Performance and evaluation of smaller local language models.
*   **Fashion-MNIST CNN:** in progress..
*   **MNIST Classifier (PyTorch):** Baseline digit classification implemented in PyTorch.
*   **MNIST Denoising Autoencoder:** Unsupervised learning for image denoising.
*   **RoBERTa vs. Qwen Sentiment:** Comparative analysis of sentiment classification using RoBERTa and Qwen models.
*   **CBIR Image Search Engine:** Content-Based Image Retrieval system for searching similar images.
